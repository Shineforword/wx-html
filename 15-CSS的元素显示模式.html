元素的显示模式

元素以什么样的方式进行显示

1.块元素:独占一行,常见块元素(div p h ul li ol )

    特点
  • 独占一行
  • 高度宽度内外边距都可以控制
  • 宽度默认是父级宽度
  • 是一个容器及盒子,可以放置行内元素,
注意:
  • 文字类内的元素不能使用块元素
  • p元素中不能再放置块元素,尤其是div
  • h1-h6也不能放置块元素

      • 2.行内元素:又称为内联元素

        特点
      1. 相邻的行内元素,一行可以显示多个
      2. 宽高直接设置是无效的
      3. 默认宽度是本身内容的高度
      4. 行内元素只能容纳文本或者其他行内元素

      3.行内块元素:同时具有块元素和行内元素的特点

        img input td
      1. 和相邻行内元素在一行上,一行可以多个(行内元素特点)
      2. 默认宽度就是其本身内容的高度(行内元素特点)
      3. 高度行高内外边距都可以控制(块元素特点)
